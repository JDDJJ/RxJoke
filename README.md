#RxJoke

----------

RxJoke是一个基于Rxjava,Retrofit,Dagger2和Mvp设计的一款App。

[More info!](http://www.jianshu.com/p/9430eca553a5 "More info")

##项目结构
 * MVP框架。
 * dagger2来完成依赖注入。
 * retrofit2+rxjava实现restful的http请求。

##第三方类库
* retrofit2
* dagger2
* butterknife
* rxjava+rxandroid
* leakcanary
* glide
* mvp
* android support design
